# 🤖 Swastik's AI Assistant

A premium, modern AI chatbot portfolio project built with **React**, **TypeScript**, and **Tailwind CSS**, powered by the **OpenAI API**.

![AI Chatbot Interface](https://via.placeholder.com/800x450?text=AI+Assistant+Interface+Screenshot)

## ✨ Features

- **🚀 Real-time Streaming**: Experience lightning-fast responses with OpenAI's streaming API.
- **🎨 Premium UI/UX**: Dark-themed, glassmorphic design inspired by modern AI interfaces.
- **💡 Suggested Prompts**: Interactive starting points for new conversations.
- **📝 Rich Markdown Support**: Full rendering of bold text, lists, and tables.
- **💻 Syntax Highlighting**: Beautifully formatted code blocks with a "Copy to Clipboard" feature.
- **⚡ Responsive Layout**: Optimized for both desktop and mobile viewing.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn
- An OpenAI API Key

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/ai-chatbot.git
   cd ai-chatbot
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Configure Environment Variables:
   Create a `.env` file in the root directory and add your OpenAI API key:
   ```env
   VITE_OPENAI_API_KEY=your_openai_api_key_here
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```
